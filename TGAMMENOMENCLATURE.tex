\section{Introduction}

\subsection{Objectif du document}
Le présent document a pour objectif de définir et formaliser les \textbf{gammes opératoires par poste de charge} ainsi que la \textbf{nomenclature correspondante} (Bill Of Materials - BOM) pour l'ensemble des produits fabriqués par CEJID pour l'année 2026.

Conformément aux spécifications, ce document doit permettre de :
\begin{itemize}
    \item Structurer les opérations de fabrication par poste de travail (découpe, usinage, placage de chants, assemblage, finition, emballage) ;
    \item Établir la liste complète des composants entrant dans chaque produit fini ;
    \item Fournir une base technique pour le calcul ultérieur des temps de cycle (TC) et des coûts de revient.
\end{itemize}

\subsection{Périmètre fonctionnel et produits concernés}
Le périmètre de ce document couvre l'ensemble des produits finis référencés dans les fichiers sources, répartis selon les catégories suivantes :

\begin{itemize}
    \item \textbf{Meubles mélaminés} (onglet \texttt{NOM MELAMINE}) : modèles INMAA, K2, INARA, DENIA, MADRID, SEBOU, AGADIR, BELFAST, ONDAS, ALICANTE, BILBAO, etc.
    \item \textbf{Meubles PVC} (onglet \texttt{PVC 2026}) : modèles MYKONOS, PATRAS, TORINO.
    \item \textbf{Meubles laqués} (onglets \texttt{LAQUEE 2026} et \texttt{LAQUEE BARDAGE}) : modèles LYON, MONACO, RABAT.
    \item \textbf{Miroirs} (onglet \texttt{NOM MELAMINE}) : modèles CORDOBA (50, 60, 70, 80, 100, 120 cm) et K3.
\end{itemize}

Pour chaque produit, sont détaillés :
\begin{itemize}
    \item La liste des composants (panneaux, quincaillerie, colles, emballages) ;
    \item Les quantités associées, exprimées dans leurs unités respectives (U, KG, M², ML) ;
    \item Les formules de calcul lorsque les quantités sont définies de manière paramétrique.
\end{itemize}

\subsection{Sources des données (fichiers Excel)}
Les données utilisées pour l'élaboration de ce document proviennent de deux fichiers Excel distincts :

\paragraph{Fichier 1 : \texttt{Nomenclature CEJID 2026.xlsx}}
Ce fichier contient plusieurs onglets :
\begin{itemize}
    \item \texttt{NOM MELAMINE} : nomenclature des meubles mélaminés et des miroirs ;
    \item \texttt{PVC 2026} : nomenclature des meubles en PVC ;
    \item \texttt{LAQUEE 2026} : nomenclature des meubles laqués ;
    \item \texttt{LAQUEE BARDAGE} : complément pour les meubles laqués avec bardage.
\end{itemize}

\paragraph{Fichier 2 : \texttt{PROCESS PRODUIT YNION.xlsx}}
Ce fichier contient les gammes opératoires sous forme de tableaux structurés par onglet :
\begin{itemize}
    \item \texttt{PROCESS PIENTURE} : gammes pour les produits laqués ;
    \item \texttt{PROCESS TIROIR LAQUEE} : gammes spécifiques pour les tiroirs laqués ;
    \item \texttt{PROCESS TIROIR MELAMINI} : gammes pour les tiroirs mélaminés ;
    \item \texttt{PROCESS PORTE MELAMINI} : gammes pour les portes mélaminées.
\end{itemize}

Chaque onglet définit les postes de charge, les opérations élémentaires, les temps associés (réglage, machine, transfert) et les références produits concernées.

\paragraph{Traitement des données}
Les quantités exprimées sous forme de formules (exemple : \texttt{=0.02+0.09+0.02}) ont été interprétées et consolidées. Les unités de mesure (U, KG, M², ML) sont conservées telles que définies dans les fichiers sources.

\subsubsection{Définition des temps opératoires}

Pour chaque poste de charge, les temps opératoires sont définis selon les paramètres suivants :

\begin{itemize}
    \item \textbf{Temps de réglage (TR)} : temps nécessaire pour préparer la machine ou le poste avant la production (en heures ou minutes)
    \item \textbf{Temps machine (TM)} : temps unitaire d'exécution de l'opération par la machine
    \item \textbf{Temps main-d'œuvre (TMO)} : temps unitaire d'intervention de l'opérateur
    \item \textbf{Temps de transfert (TT)} : temps de déplacement entre deux postes successifs
\end{itemize}

\subsubsection{Règles de conversion et d'arrondi}

\begin{itemize}
    \item Les surfaces de panneaux sont calculées à partir des dimensions nominales (ex: 0,54 M² pour un panneau de dimensions spécifiques).
    \item Les longueurs de bandes de chant sont exprimées en mètres linéaires avec une précision de 0,01 ML.

\end{itemize}
